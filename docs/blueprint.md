# **App Name**: Flo-Pro (Crafted 4UR Convenience)

## Core Features:

- Canvas Editor: Interactive drag-and-drop interface for creating flowcharts.
- Symbol Library: A set of common flowchart shapes, such as rectangles, diamonds, ovals, and arrows, all customizable and easy to insert.
- Color Palette: Allows users to change colors of individual flowchart elements as well as connector lines.
- Smart Suggestion: Based on the flowchart you are currently drawing, suggest the most appropriate next symbol to add to the chart. The tool uses the current node and the relationships to other nodes as context.
- Export Functionality: Enables export to common formats such as PNG, JPEG, and PDF for sharing and documentation purposes.

## Style Guidelines:

- Primary color: Vivid blue (#4285F4), reminiscent of brainstorming and clarity.
- Background color: Very light blue (#E3F2FD), creates a clean, calming backdrop for focus.
- Accent color: Complementary violet (#8E44AD) to highlight selected elements or call-to-action buttons.
- Clean and modern font that supports diverse diagram content clearly.
- Simple, universally recognizable symbols for flowchart shapes and actions.
- Whitespace-centric layout to focus attention and provide breathing room for flowchart diagrams.
